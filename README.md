# myJson

## 项目简介
myJson是一个基于C++实现的json的配置文件解析工具，支持json文件解析，支持json文件生成。

## Json 数据格式

JSON 是一种数据交换格式，它基于 JavaScript 对象表示法（JavaScript Object Notation，简称 JSON），但 JSON 语法比 JavaScript objects 更简单。



可以表示的数据类型包括：
- 数字（整数、浮点数）
- 字符串
- 布尔值（true、false）
- 数组
- 对象（键值对的集合）
- null
